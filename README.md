# RetailPulse Analytics

## Enterprise Retail Data Visualization & Business Analytics Project

---

# Project Overview

RetailPulse Analytics is an end-to-end retail data analytics and visualization project built using Python.

The project simulates a real multinational retail business environment and focuses on transforming raw retail data into meaningful business insights through:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- KPI Reporting
- Professional Data Visualization

This project demonstrates practical skills in:
- Data Cleaning
- Data Analysis
- Business Intelligence
- Data Visualization
- KPI Reporting
- Retail Analytics

# Dataset Features

The dataset simulates a realistic enterprise-level retail business with:
- 100,000+ retail transaction records
- Multinational sales data
- Customer demographics
- Product categories
- Shipping information
- Payment methods
- Profit & discount analysis

### Real-World Data Challenges Included

- Missing values
- Duplicate records
- Inconsistent category naming
- Invalid customer ages
- Negative profit transactions
- Inconsistent payment method spellings

---

# Data Cleaning Process

The dataset was cleaned using Python and Pandas by:
- Handling missing values
- Removing duplicate records
- Fixing invalid customer ages
- Converting date columns
- Standardizing inconsistent values
- Preparing data for visualization

# Key Business Insights

- Electronics category generated the highest overall revenue.
- Clothing products experienced the highest return rates.
- Large discounts negatively impacted profitability.
- Consumer customer segment contributed maximum sales.
- Digital payment methods dominated customer transactions.
- Sales and profit showed strong positive correlation.
